Yeti CMS Example

Yeti CMS is a Sitebuilder-Style CMS that treats your static site like a simple collection of HTML documents that you can edit. Drag and drop your way to the website you want, from any template.

SEE IT IN ACTION

About

Free and Open Source, the Yeti CMS delivers features once only available in premium sitebuilder products.

Using the Demo

You can right-click on most things to edit them. Saving is disabled for the demo.

Join the fun!

Features

Yeti brings all the features you need to maintain your website,
no matter which tools you used to build it, or when.

Responsive Editor

Yeti's minimalistic editor works on the device you're working on, so that you can see your changes instantly, no matter which device you're on.

Web Compliant

Yeti CMS works with whatever web-compliant technologies you throw at it. Javascript libraries? HTML5? CSS3? No problem!

No Templates

All you need to get started is an HTML file & one line of Javascript. With Yeti, there are no messy templating languages to learn.

Free and Open Source

Yeti CMS is maintained by an awesome team of contributers. Want to extend Yeti CMS? Submit a Pull Request!
